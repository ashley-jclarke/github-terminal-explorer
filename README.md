# github-terminal-explorer

Search for git repositories from your terminal.

![Preview of github terminal explorer showing search for i3-wm with results in a table showing data: Repository name, author, description and number of stars](screenshots/image.0.png)

## Current Features

+ Search for repositories
+ Check repository description, number of stars, owner

## Future Features

+ Git clone from tui
+ Preview README.md of viewed repository
+ Get information on authors
+ File Explorer for raw repository files
+ Issue Explorer
